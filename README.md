# curriculum-vitae

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Build a pdf

```sh
npm run build
npm run build-pdf
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
